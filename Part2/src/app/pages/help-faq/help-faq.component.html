Troubleshooting Guide

Common Issues & Solutions

  • Error: "Item ID already exists"

    Solution: Choose a unique Item ID that doesn't already exist in the system.

  • Error: "Negative values not allowed"

    Solution: Quantity and Price fields must be non-negative numbers. Enter 0 or higher.

  • Page not rendering correctly

    Solution: Clear browser cache or use incognito mode. Ensure you're using a modern browser (Chrome, Firefox, Edge).

  • Search not working

    Solution: Make sure you're typing in the search box on the Search page. The search is case-insensitive but requires at least one character.

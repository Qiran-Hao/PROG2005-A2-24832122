## Project Overview
This project is a fully functional Inventory Management System developed for PROG2005 Assessment 2.  
It includes two parts: Vanilla TypeScript (Part 1) and Angular 18 (Part 2).

---

## Part 1 – Vanilla TypeScript
Features:
- Add, update, delete, search inventory items
- Responsive UI design
- Strict TypeScript validation
- Compiled to JavaScript for browser execution

How to run:
1. Open Part1 folder
2. Run tsc to compile
3. Open index.html in browser

---

## Part 2 – Angular 18
Features:
- 5 complete pages: Home, Inventory Manage, Search Filter, Privacy & Security, Help & FAQ
- Full CRUD operations
- Multi-condition search and filter
- Professional UI & responsive layout
- Complete routing system

How to run:
1. Open Part2 folder
2. Run npm install
3. Run ng serve
4. Open http://localhost:4200
